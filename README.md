# Expense Manager API

This is the backend server for the Expense Manager application.

## Repository

The server code is hosted at:
[https://github.com/Rubayet-hasan-yasin/expense-manager-apis](https://github.com/Rubayet-hasan-yasin/expense-manager-apis)

## Getting Started

1. Install dependencies (e.g. `npm install` or `yarn install`)
2. Configure `.env`
3. Run the development server
